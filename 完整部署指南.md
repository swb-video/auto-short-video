# 全自动短视频变现系统 - 完整部署指南

## 前置要求

### 1. 安装Git
1. 访问 https://git-scm.com/download/win
2. 下载并安装Git（一路点击Next即可）
3. 安装完成后，打开CMD输入 `git --version` 验证

---

## 部署步骤

### 步骤1：打开CMD
按 `Win + R`，输入 `cmd`，回车

### 步骤2：进入项目目录
```cmd
cd /d c:\Users\41337\WorkBuddy\20260319083955
```

### 步骤3：初始化Git
```cmd
git init
```

### 步骤4：配置Git用户信息
```cmd
git config user.name "swb-video"
git config user.email "deploy@auto-short.video"
```

### 步骤5：添加所有文件
```cmd
git add .
```

### 步骤6：提交代码
```cmd
git commit -m "Initial commit"
```

### 步骤7：创建main分支
```cmd
git branch -M main
```

### 步骤8：连接GitHub仓库
```cmd
git remote add origin https://github.com/swb-video/auto-short-video.git
```

### 步骤9：推送代码
```cmd
git push -u origin main
```

**如果推送失败**，请先在浏览器中登录GitHub，然后重新执行步骤9。

## 启动自动化

1. 访问 https://github.com/swb-video/auto-short-video/actions
2. 点击左侧 **「Auto Short Video Workflow」**
3. 点击 **「Run workflow」** → 选择 **「crawl-hotspots」**
4. 点击 **「Run workflow」** 运行首次测试

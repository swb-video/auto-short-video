# 飞书应用创建步骤（3分钟完成）

## 步骤1：创建应用

1. 访问 https://open.feishu.cn/app
2. 点击 **「创建企业自建应用」**
3. 填写信息：
   - 应用名称：`短视频自动化`
   - 应用描述：`AI短视频内容自动化系统`
   - 点击 **「创建」**

## 步骤2：获取凭证

创建后自动进入应用管理页面：

1. 在 **「凭证与基础信息」** 页面
2. 复制以下信息（保存到记事本）：
   - **App ID**：`cli_xxxxxxxx`
   - **App Secret**：点击显示并复制

## 步骤3：添加权限

1. 左侧菜单点击 **「权限管理」**
2. 点击 **「申请权限」**，搜索并添加：
   - ✅ `bitable:record`（读写多维表格）
   - ✅ `bitable:view`（查看多维表格）
   - ✅ `im:chat:send`（发送消息）
3. 点击 **「批量申请」**

## 步骤4：创建表格

1. 打开飞书，创建 **「多维表格」**
2. 表格名称：`短视频选题库`
3. 添加以下字段：

| 字段名 | 类型 | 选项（单选类型需要）|
|--------|------|-------------------|
| 标题 | 文本 | - |
| 热度值 | 数字 | - |
| 来源平台 | 文本 | - |
| 链接 | 文本 | - |
| 匹配关键词 | 文本 | - |
| 匹配分数 | 数字 | - |
| 抓取时间 | 日期时间 | - |
| 处理状态 | 单选 | 待审核、脚本已生成、已发布、已忽略 |
| 脚本标题 | 文本 | - |
| 脚本内容 | 文本 | - |
| 标签 | 文本 | - |
| 生成时间 | 日期时间 | - |

## 步骤5：获取表格Token

1. 打开刚创建的表格
2. 点击右上角 **「...」** → **「复制链接」**
3. 链接格式示例：
   ```
   https://example.feishu.cn/base/AbCdEfGh?table=IjKlMnOp
   ```
4. 提取：
   - **App Token**：`base/` 后面的 `AbCdEfGh`
   - **Table ID**：`table=` 后面的 `IjKlMnOp`

## 步骤6：创建群机器人

1. 在飞书创建一个群（或选择已有群）
2. 点击群右上角 **「设置」** → **「群机器人」**
3. 点击 **「添加机器人」** → **「自定义机器人」**
4. 名称：`短视频日报`
5. 复制 **Webhook地址**

## 步骤7：发布应用

1. 回到 https://open.feishu.cn/app
2. 点击 **「版本管理与发布」**
3. 点击 **「创建版本」**
4. 填写版本号：`1.0.0`
5. 点击 **「申请发布」**
6. 等待管理员审批（如果是自己的企业，直接通过）

---

## 需要保存的信息

完成以上步骤后，您应该有：

| 信息 | 格式示例 |
|------|----------|
| App ID | `cli_xxxxxxxx` |
| App Secret | `xxxxxxxx` |
| App Token | `AbCdEfGh` |
| Table ID | `IjKlMnOp` |
| Webhook | `https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx` |

**请将这些信息保存好，下一步需要用到！**
